## 搭建 V2Ray / Xray 节点：一条命令的事

买好 VPS 之后，正式进入搬瓦工科学上网教程的核心部分。

### 前置工具准备

**SSH 工具**：Windows 推荐用 MobaXterm 或者系统自带的 PowerShell（Windows 10 以后原生支持 ssh 命令）；macOS 直接用终端。

登录命令格式是：

bash
ssh root@你的IP地址 -p 你的SSH端口号


IP 地址和端口在邮件里和 KiwiVM 后台都能看到。密码第一次登录前需要在 KiwiVM 里点 Root password modification 重置一次，然后复制粘贴进去（输入时不显示，粘贴完直接回车）。

### 安装推荐系统

KiwiVM 后台可以一键装系统，推荐选 **Debian 12**。搬瓦工在部分系统上默认开启了 BBR 加速，Debian 12 的配套比较完整。

### 一键安装 V2Ray / Xray

登录进服务器之后，运行下面这条命令（233boy 的开源脚本，用的人多，坑也少）：

bash
bash <(wget -qO- -o /dev/null yums.sh) xray


或者直接安装 V2Ray 版本：

bash
bash <(wget -qO- -o /dev/null yums.sh) v2ray


脚本跑起来会让你选协议，新手直接回车默认用 VMess 就行。

安装完之后脚本会输出一段配置信息，包括服务器地址、端口、UUID、加密方式。

**把这些信息截图或者记下来**，后面客户端要用到。

### 开启 BBR 加速

如果安装好之后速度不理想，检查一下 BBR 有没有开：

bash
sysctl net.ipv4.tcp_congestion_control


输出 `net.ipv4.tcp_congestion_control = bbr` 就是已经开启了。如果没开，搬瓦工 KiwiVM 后台有一键启用 BBR 的选项，点一下重启就行。

用下来的体验是，BBR 开启前后在晚高峰的速度差距还是比较明显的，尤其是看视频的时候。

## IP 被封了怎么办

这个问题几乎每个用搬瓦工科学上网的人都会遇到。

IP 被封的判断方法：用手机 4G（不走 WiFi）ping 一下服务器 IP，ping 不通基本可以确认。或者用在线工具分别测国内和国外节点的连通性，国内全超时、国外正常，就是被墙了。

搬瓦工 KiwiVM 后台有换 IP 服务，收费大概 8 美元左右，提交申请之后几小时内处理完，新 IP 邮件通知。换完之后重新配置客户端用新 IP 就行，服务器上的代理软件不用重装。

预防 IP 被封的几个经验：协议选 VMess 或者 VLESS+Reality（比纯 TCP 流量伪装更好）；不要用特别明显的端口；流量别搞得太大太集中。
